About Azhagu
============
Azhagu means beauty in Tamil language. This is a single and two-column 
responsive theme, compatible for desktops, tablets, phones and other devices.

Features
========
  Responsive Layout.
  Mobile Friendly.
  Configurable Social Block for RSS Feed and Facebook, Twitter, Google Plus 
  and Pintrest.
  Sass Compass Support
  1 Column and 2 Columns layout
  3 Columns Footer.
  Total of 9 Regions
  Compatible and tested on Firefox, Chrome browsers and Safari

Browser compatibility:
=====================
The theme has been tested on following browsers. Firefox, Chrome browsers 
and Safari

Drupal compatibility:
=====================
This theme is compatible with Drupal 7.x.x
